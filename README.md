# Trabalho Escolar — Site Temático

Site HTML desenvolvido para ajudar minha cunhada em um trabalho escolar.

## Sobre

Site estático com quatro seções de conteúdo distribuídas em duas páginas, além de um quiz interativo separado.

**Temas abordados:**
- Filme Bohemian Rhapsody
- Guerra Irã e Estados Unidos e o Petróleo
- O Oscar e sua Importância
- Educação Brasileira — Erros e Acertos

## Estrutura de arquivos

```
/
├── index.html      # Página 1 — Bohemian Rhapsody e Irã & EUA
├── pagina2.html    # Página 2 — Oscar e Educação Brasileira
├── quiz.html       # Quiz interativo com 8 perguntas
├── style.css       # Estilos compartilhados entre todas as páginas
└── script.js       # Lógica do quiz
```

## Tecnologias

- HTML5
- CSS3
- JavaScript vanilla

## Como rodar

Abra o arquivo `index.html` no navegador. Sem dependências, sem instalação.
